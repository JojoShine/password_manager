## 1. 安装与首次设置

### 1.1 系统要求

**桌面端**
- macOS 10.14+ / Windows 10+ / Linux (Ubuntu 18.04+)
- 最少 100MB 存储空间
- 2GB 内存

**移动端**
- Android 6.0+
- 50MB 存储空间

### 1.2 下载安装

#### 方式一：从 GitHub Releases 下载
1. 访问 [Releases页面](https://github.com/JojoShine/password_manager/releases)
2. 根据系统下载对应版本：
   - **macOS**: `PasswordManager-macOS.dmg`
   - **Windows**: `PasswordManager-Windows.zip`
   - **Linux**: `PasswordManager-Linux.AppImage`
   - **Android**: `PasswordManager-Android.apk`

#### 方式二：从源码构建
```bash
git clone https://github.com/JojoShine/password_manager.git
cd password_manager
flutter pub get
flutter build [platform] --release
```

### 1.3 首次启动设置

#### 步骤1：创建主密码
1. 启动应用后，您将看到登录界面
2. 点击"创建主密码"按钮
3. 设置一个强密码（建议8位以上，包含大小写字母、数字和符号）
4. 确认密码并牢记（无法找回！）

> ⚠️ **重要提醒**：主密码无法找回，如果忘记将无法访问您的数据

#### 步骤2：生物识别设置（可选）
1. 在设置界面中选择"安全设置"
2. 开启"生物识别解锁"
3. 按照系统提示设置Touch ID或Face ID

#### 步骤3：基本配置
1. 选择主题模式（明亮/深色/自动）
2. 设置自动锁定时间
3. 配置安全选项

### 2.2 添加密码

#### 方法一：手动添加
1. 点击右上角"+"按钮
2. 填写必要信息：
   - **标题**：易于识别的名称（如"Gmail账户"）
   - **用户名**：登录用户名或邮箱
   - **密码**：账户密码
   - **网站**：网站地址（可选）
   - **分类**：选择合适的分类
   - **备注**：额外信息（可选）
3. 点击"保存"完成添加

#### 方法二：使用密码生成器
1. 在添加界面点击"生成密码"按钮
2. 配置密码参数：
   - **长度**：6-50位（推荐16位以上）
   - **包含字符**：大写字母、小写字母、数字、符号
   - **排除相似字符**：避免 i、l、1、O、0 等易混淆字符
3. 点击"生成"获得安全密码
4. 可选择直接保存为新条目

### 2.3 管理密码

#### 查看和编辑
- **查看密码**：点击密码条目，密码将自动复制到剪贴板
- **编辑条目**：双击条目或右键选择"编辑"
- **删除条目**：右键选择"删除"并确认

#### 搜索和筛选
- **快速搜索**：在顶部搜索框输入关键词
- **分类筛选**：点击分类标签筛选对应类型
- **多字段搜索**：支持搜索标题、用户名、网站等

#### 批量操作
1. **多选模式**：长按任意条目进入多选模式
2. **批量删除**：选择多个条目后点击删除按钮
3. **批量导出**：选择条目后选择导出选项

### 2.4 密码安全分析

#### 安全报告
1. 进入"设置" → "安全分析"
2. 查看安全评分和建议
3. 处理以下问题：
   - **弱密码**：长度短或复杂度低的密码
   - **重复密码**：多个账户使用相同密码
   - **过期密码**：长时间未更新的密码

#### 密码强度指标
- 🔴 **弱**：不安全，建议立即更换
- 🟡 **中**：一般，建议优化
- 🟢 **强**：安全，可放心使用

## 3. 浏览器扩展使用指南

### 3.1 扩展安装

#### 支持的浏览器
- Google Chrome
- Microsoft Edge
- 其他基于 Chromium 的浏览器

#### 安装步骤
1. **下载扩展包**
   - 在主应用中点击"浏览器扩展"
   - 点击"立即下载"获取扩展文件
   - 解压到固定文件夹（不要删除）

2. **加载到浏览器**
   - 打开浏览器扩展管理页面：
     - Chrome: `chrome://extensions/`
     - Edge: `edge://extensions/`
   - 开启"开发者模式"开关
   - 点击"加载已解压的扩展程序"
   - 选择解压后的扩展文件夹
   - 扩展图标将出现在工具栏

### 3.2 扩展使用

#### 首次配置
1. 确保密码管理器主应用已启动
2. 点击扩展图标进行连接测试
3. 如连接成功，扩展即可正常使用

#### 自动填充功能
1. **访问登录页面**：打开需要登录的网站
2. **检测登录表单**：扩展会自动检测用户名和密码字段
3. **点击扩展图标**：选择要填充的账户
4. **自动填充**：账户信息将自动填入表单

#### 保存新密码
1. **检测新密码**：在注册或修改密码时，扩展会提示保存
2. **确认保存**：点击提示框中的"保存"按钮
3. **填写信息**：完善标题、分类等信息
4. **同步到主应用**：新密码将自动同步

### 3.3 扩展设置

#### 连接配置
- **本地端口**：默认为 8080，可在主应用设置中修改
- **连接状态**：绿色表示连接正常，红色表示连接失败

#### 安全选项
- **自动清除剪贴板**：填充后自动清空剪贴板
- **HTTPS检测**：仅在HTTPS网站上启用自动填充
- **域名匹配**：严格匹配网站域名

## 4. 数据管理

### 4.1 数据备份

#### 加密备份（推荐）
1. 进入"设置" → "数据管理" → "导出数据"
2. 选择"加密导出"
3. 输入导出密码（可与主密码不同）
4. 选择保存位置
5. 妥善保存导出文件和密码

#### 明文备份
1. 选择"明文导出"
2. 选择导出格式：
   - **JSON**：完整数据，支持重新导入
   - **CSV**：表格格式，可用Excel打开
3. 注意：明文文件包含敏感信息，请安全存储

### 4.2 数据导入

#### 从备份恢复
1. "设置" → "数据管理" → "导入数据"
2. 选择备份文件
3. 如果是加密文件，输入导出时设置的密码
4. 选择导入方式：
   - **覆盖现有数据**：清空当前数据，导入备份
   - **合并数据**：保留现有数据，添加导入的数据

#### 从其他密码管理器导入
支持的格式：
- **Chrome 密码导出**：CSV格式
- **Firefox 密码导出**：CSV格式
- **LastPass 导出**：CSV格式
- **1Password 导出**：CSV格式

### 4.3 数据同步

#### 本地同步
- 所有数据存储在本地设备
- 通过导出/导入实现设备间同步
- 推荐使用加密备份确保安全

#### 云同步（规划中）
- 端到端加密的云同步服务
- 支持多设备实时同步
- 零知识架构保护隐私

## 5. 安全与隐私

### 5.1 安全机制

#### 数据加密
- **AES-256-GCM**：军用级加密算法
- **PBKDF2-SHA256**：密钥派生函数
- **HMAC验证**：确保数据完整性

#### 认证保护
- **主密码**：使用强哈希算法存储
- **生物识别**：支持Touch ID/Face ID
- **自动锁定**：设定时间后自动锁定

#### 本地存储
- **零云存储**：所有数据仅存储在本地
- **沙盒保护**：应用数据与系统隔离
- **安全删除**：彻底清除已删除数据

### 5.2 隐私保护

#### 数据收集承诺
- ✅ **零数据收集**：不收集任何个人信息
- ✅ **无网络传输**：密码数据不上传
- ✅ **开源透明**：源码公开接受审查
- ✅ **本地运行**：完全离线工作

#### 防护措施
- **防截屏**：在敏感界面禁用截屏
- **防拍照**：相机无法访问应用界面
- **内存保护**：敏感数据加密存储在内存中

### 5.3 安全建议

#### 主密码设置
- **长度**：至少8位，推荐12位以上
- **复杂性**：包含大小写字母、数字、符号
- **唯一性**：不要与其他账户密码相同
- **定期更换**：建议6个月更换一次

#### 使用习惯
- **及时锁定**：离开时及时锁定应用
- **定期备份**：定期导出加密备份
- **安全环境**：在安全的网络环境下使用
- **及时更新**：保持应用版本最新

## 7. 常见问题

### 7.1 账户管理

**Q: 忘记主密码怎么办？**
A: 很遗憾，主密码无法找回。这是零知识架构的安全特性，即使开发者也无法访问您的数据。建议：
- 使用生物识别减少输入频率
- 定期备份数据到安全位置
- 使用密码提示（不要太明显）

**Q: 如何更改主密码？**
A: 1. 进入"设置" → "安全设置" → "更改主密码"
2. 输入当前主密码验证身份
3. 设置新的主密码
4. 所有数据将使用新密码重新加密

### 7.2 数据安全

**Q: 数据存储在哪里？**
A: 所有数据都存储在您的设备本地：
- **Windows**: `%APPDATA%\PasswordManager\`
- **macOS**: `~/Library/Application Support/PasswordManager/`
- **Linux**: `~/.config/PasswordManager/`
- **Android**: 应用私有目录（其他应用无法访问）

**Q: 卸载应用后数据会丢失吗？**
A: 是的，卸载应用会删除所有本地数据。请在卸载前：
1. 导出数据备份
2. 确认备份文件完整可用
3. 妥善保存备份文件和密码

### 7.3 浏览器扩展

**Q: 扩展无法连接到主应用？**
A: 请检查：
1. 主应用是否正在运行
2. 本地服务器是否启动（设置中查看）
3. 防火墙是否阻止了连接
4. 端口是否被其他程序占用

**Q: 扩展不能自动填充？**
A: 可能原因：
1. 网站使用了复杂的表单结构
2. 网站域名与保存的不匹配
3. 网站禁用了自动填充
4. 浏览器安全策略限制

### 7.4 性能问题

**Q: 应用启动缓慢？**
A: 优化建议：
1. 定期清理无用的密码条目
2. 减少过多的标签和分类
3. 关闭不必要的后台进程
4. 更新到最新版本

**Q: 搜索速度慢？**
A: 当密码条目过多时：
1. 使用更精确的搜索关键词
2. 利用分类筛选缩小范围
3. 定期整理和删除无用条目

## 8. 故障排除

### 8.1 无法启动

#### 症状：应用无法正常启动
**解决方案：**
1. **检查系统要求**：确认系统版本满足最低要求
2. **重新安装**：下载最新版本重新安装
3. **清除配置**：删除配置文件夹后重新启动
4. **查看日志**：检查系统日志了解错误信息

#### 症状：启动后立即崩溃
**解决方案：**
1. **兼容性模式**：在兼容性模式下运行
2. **显卡驱动**：更新显卡驱动程序
3. **杀毒软件**：暂时关闭杀毒软件测试
4. **管理员权限**：以管理员权限运行

### 8.2 数据问题

#### 症状：数据丢失或损坏
**解决方案：**
1. **检查备份**：查看是否有自动备份文件
2. **数据恢复**：尝试从备份文件恢复
3. **修复数据库**：使用内置的数据修复功能
4. **联系支持**：如问题严重，请联系技术支持

#### 症状：导入失败
**解决方案：**
1. **文件格式**：确认导入文件格式正确
2. **文件编码**：确保文件使用UTF-8编码
3. **数据格式**：检查CSV文件的列标题是否正确
4. **分步导入**：尝试分批导入大量数据

### 8.3 网络连接

#### 症状：浏览器扩展连接失败
**解决方案：**
1. **端口检查**：确认本地服务器端口未被占用
2. **防火墙设置**：添加应用到防火墙白名单
3. **权限检查**：确保应用有网络访问权限
4. **重启服务**：重启主应用的本地服务器
